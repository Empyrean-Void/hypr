## Installation (Nvidia)

- Install the needed dependencies

```
yay -S hyprland-nvidia hyprpaper waybar-hyprland xdg-desktop-portal-hyprland-git
```

- Clone this repository

```
git clone https://github.com/Empyrean-Void/hypr.git ~/.config/hypr/
```
